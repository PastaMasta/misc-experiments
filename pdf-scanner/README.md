pdf-scanner
------------

Install deps:
```
pip3 install -r ./requirements.txt
```

Run with:
```
./scan-pdf.py ./*.pdf
```

Started looking at this from:
https://www.analyticsvidhya.com/blog/2021/06/data-extraction-from-unstructured-pdfs/
